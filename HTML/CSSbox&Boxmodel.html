블록 레벨 요소

  • 요소를 삽입했을 때 혼자 한 줄을 차지하는 요소
  • 요소의 너비가 100%
  • ex) div, p 등

인라인 레벨 요소

  • 줄을 차지하지 않는 요소
  • 화면에 표시되는 컨텐츠만큼만 영역을 차지하고 나머지 공간에는 다른 요소가 올 수 있음
  • ex) img, strong 등

박스 모델

  • 실제 컨텐츠 영역, 패딩(padding), 박스의 테두리(border), 마진(margin)등의 요소로 구성

width, height 속성

  • 실제 콘텐츠 영역의 크기 지정
  • ex) width or height : 크기 or 백분율 or auto
  • 크기 : 너비나 높이 값을 px이나 cm 같은 단위와 함께 수치로 지정
  • 백분율 : 박스 모델을 포함하는 부모 요소를 기준으로 너비나 높이 값을 백분율로 지정
  • auto : 박스 모델의 너비와 높이 값이 콘텐츠 양에 따라 자동으로 결정. 기본값

ex) width : 200px; height : 100px; margin : 5px; background : blue

ex) width : 50%; height : 100px; margin : 5px; background : blue

display 속성

  • 블록 레벨 요소를 인라인 레벨 요소로 바꾸거나 인라인 레벨 요소를 블록 레벨 요소로 바꿈
  • ex) display : none | contents | block | inline | inline-block | table | table-cell 등
    1. display:block
      • 해당 요소를 블록 레벨로 지정
    2. display:inline
      • 블록 레벨 요소를 인라인 레벨로 지정
      • display:
      • inline
      • 예제
    3. display:inline-block
      • 요소를 인라인 레벨로 배치하면서 내용에는 블록 레벨 속성을 지정
      • display:
      • inline-block
      • 예제
    4. display:none
      • 해당 요소를 화면에 표시하지 않고, 화면에서 공간도 차지하지 않음
      • display: none 예제
