스타일 형식 : 선택자 { 스타일 속성: 속성 값 }

  • 선택자 : { 와 } 사이에 정의한 스타일 규칙이 적용될 대상
  • 스타일 속성과 속성 값 : '속성: 속성 값'과 같은 형식으로 함께 표시.
    여러개일 경우 세미콜론으로 구분

스타일 표기 방법

  • 선택자 { 스타일 속성: 속성 값 }
  • 선택자 {
            스타일 속성: 속성 값
    }
            
  • 선택자
    {
        스타일 속성: 속성 값
    }

스타일 주석

  • /* 와 */ 사이에 주석 내용 입력
  • 한 줄 또는 여러 줄을 입력 가능

내부 스타일 시트

  • 웹 문서 안에서 사용할 스타일을 문서 안에 정리한 것
  • 모든 스타일 정보는 <head>태그와 </head>태그 안에서 정의
  • <style>태그와 </style>태그 사이에 작성

외부 스타일 시트(외부 스타일 시트에서 적용 중)

  • 여러 웹 문서에서 사용할 스타일을 별도 파일로 저장해 놓고 필요할 때마다 가져와 사용
  • <style>태그 없이 <head>태그 안에 <link>태그만 사용해 미리 만들어 놓은 외부 스타일 시트 파일 연결
  • ex) <link href="Style.css" rel="stylesheet" type="text/css">

인라인 스타일(적용 중)

  • 스타일 시트를 사용하지 않고 스타일을 적용할 대상에 직접 표시
  • 적용하고 싶은 태그에 style 속성을 사용해 "style="속성: 속성 값;" 형태로 스타일 적용
  • ex) <ul style="color:goldenrod"></ul>

전체 선택자(외부 스타일 시트에서 적용 중)

  • 페이지에 있는 모든 요소를 대상으로 스타일을 적용할 때 사용
  • 다른 선택자와 함께 모든 하위 요소에 한꺼번에 스타일을 적용하려고 할 때 주로 사용.
  • ex) * { 속성: 속성 값; ... }

태그 선택자

  • 문서에서 특정 태그를 사용한 모든 요소에 스타일이 적용
  • ex) 태그 { 속성: 속성 값; ...}

클래스(class) 선택자, id 선택자(외부 스타일 시트에서 적용 중)

  • 공통점 : 요소의 특정 부분에만 스타일 적용
  • 차이점
    • 클래스 선택자 : 문서 안에서 여러 번 반복할 스타일이라면 클래스 선택자로 정의
      마침표(.) 다음에 클래스 이름 지정
    • ex) .classname
        {
            color:blue;
        }
    • id 선택자 : 문서 안에서 한번만 사용한다면 id 선택자로 정의
      파운드(#) 다음에 id 이름 지정
    • ex) #id
        {
            color:blue;
        }
