문단 : text-indent 속성

  • 문단의 첫 글자를 얼마나 들여 쓸지 지정(크기, 백분율)
  • ex) text-indent: 15px 적용

문단 : line-height 속성

  • 문단의 줄 간격 지정(숫자, 크기, 백분율)
  • 숫자와 백분율은 부모 요소를 기준으로 몇 배인지 지정
  • 보통 글자 크기의 1.5~2배가 적당
  • ex) line-height: 2 적용

목록 : list-style-type 속성

  • 순서 없는 목록의 불릿이나 순서 목록의 숫자를 바꾸는 속성
    1. 순서 없는 목록의 불릿 바꾸기
      • dise(●) : 채운 원(기본값)
      • circle(○) : 빈 원
      • square(■) : 채운 사각형
      • none : 불릿 없애기
    2. 순서 목록의 숫자 바꾸기
      1. decimal : 1로 시작하는 십진수(기본 값)
      2. decimal-leading-zero : 앞에 0이 붙는 십진수
      3. lower-roman : 소문자 로마 숫자
      4. upper-roman : 대분자 로마 숫자
      5. lower-alpha : 소문자 알파벳
      6. lower-latin : 소문자 알파벳
      7. upper-alpha : 대문자 알파벳
      8. upper-latin : 대문자 알파벳
      9. armenian : 아르메니아 숫자
      10. georgian : 조지 왕조시대의 숫자

목록 : list-style-image 속성

  • 순서 없는 목록의 불릿을 이미지로 바꾸는 속성
  • ex) list-style-image: url('url')

목록 : list-style-position 속성

  • 불릿이나 번호를 들여쓰거나 내어쓸 수 있음
  • inside : 불릿이나 숫자를 안쪽으로 들여 씀
  • outside : 불릿이나 숫자를 밖으로 내어 씀(기본값)

목록 : list-style 속성

  • list-style-type, list-style-position, list-style-image 속성을 묶어서 표현
  • ex) list-style: circle, inside
  • ex2) list-style: inside, url('url')
