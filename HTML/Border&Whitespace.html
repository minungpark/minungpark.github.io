border-style 속성

  • 기본값이 none : 화면에 테두리 표시 안됨
  • 테두리를 그리기 위해서는 맨 먼저 테두리 스타일부터 지정
  • 각 방향별로 지정 가능(border-top-style, border-right-style, border-bottom-style, border-left-style)
  • none : 기본 값. 테두리가 나타나지 않음
  • hidden : 테두리가 나타나지 않음. border-collapse:collapse 일 경우, 다른 테두리도 표시되지 않음
  • dashed : 테두리를 짧은 선(직선으로 된 점선)으로 표시
  • dotted : 테두리를 점선으로 표시
  • double : 테두리를 이중선(겹선)으로 표시, 선 사이의 간격은 border-width로 지정
  • groove : 테두리를 창에 조각한 것처럼 표시. 홈이 파인듯 입체적으로 보임
  • inset : border-collapse:separate 일 경우, 전체 박스 테두리가 창에 박혀 있는 것처럼 표시.
    border-collapse:collapse 일 경우, groove와 동일
  • outset : border-collapse:separate 일 경우, 전체 박스 테두리가 창에서 튀어나온 것처럼 표시.
    border-collapse:collapse 일 경우, ridge와 동일
  • ridge : 테두리를 창에서 튀어나온 것처럼 표시
  • solid : 테두리를 실선으로 표현

border-width 속성

  • 테두리 두께 지정
  • 각 방향별로 지정 가능(border-top-width, border-right-width, border-bottom-width, border-left-width)
  • 크기, thin(얇게), medium(중간), thick(두껍게) 사용 가능
    1. 속성 값이 1개 : 네 방향에 모두 같은 두께
    2. 속성 값이 2개 : 위아래, 좌우 묶어서
    3. 속성 값이 3개 : top → right → bottom. left 값은 마주보는 변인 right 값으로 지정
    4. 속성 값이 4개 : top → right → bottom → left

border-color 속성

  • 테두리 색상 지정
  • 각 방향별로 지정 가능(border-top-color, border-right-color, border-bottom-color, border-left-color)
    1. 속성 값이 1개 : 네 방향에 모두 같은 두께
    2. 속성 값이 2개 : 위아래, 좌우 묶어서
    3. 속성 값이 3개 : top → right → bottom. left 값은 마주보는 변인 right 값으로 지정
    4. 속성 값이 4개 : top → right → bottom → left

border 속성

  • 테두리 스타일과 두께, 색상 등을 묶어 표시
  • 순서는 상관 없음
  • 각 방향별로 지정 가능(border-top, border-right, border-bottom, border-left)
  • ex) border : 5px solid red

border-radius 속성

  • 박스 모서리 부분을 둥글게 처리
  • 각 모서리 별로 지정 가능(border-top-left-radisu, border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius)
  • 크기, 백분율 사용 가능
  • 가로 반지름 크기, 세로 반지름 크기 지정 시 타원형식
  • ex) border-radius: 20px
  • ex) border-radius: 5px 10px

box-shadow 속성

  • 선택한 요소에 그림자 효과 내기
  • 수평 거리 : 그림자의 수평 옵셋 거리(수평으로 얼마나 떨어져 있는지). 양수는 오른쪽, 음수는 왼쪽. 필수 값
  • 수직 거리 : 그림자의 수직 옵셋 거리(세로로 얼마나 떨어져 있는지). 양수는 요소의 아래쪽, 음수는 요소의 위쪽. 필수 값
  • 흐림 정도 : 그림자의 흐림 정도(blue radius)를 지정. 이 값을 생략하면 0을 기본값으로 해 진한 글미자 표시.
    이 값이 커질수록 부드러운 그림자 표시 가능. 음수값 사용 불가
  • 번짐 정도 : 그림자의 번지는 정도를 나타냄. 양수는 그림자가 모든 방향으로 퍼져 나가므로 그림자가 박스보다 크게 표시.
    음수는 그림자가 모든 방향으로 축소되어 보임. 기본값은 0
  • 색상 : 그림자의 색상 지정. 한 가지만 지정 또는 여러개의 색상 지정 가능. 기본값은 현재 글자 색.
  • inset : 이 키워드를 함께 표시하면 안쪽 그림자로 나타남.
  • ex) box-shadow : 5px 3px 10px 3px blue

padding 속성

  • 컨텐츠 영역과 테두리 사이의 여백
  • 각 방향별로 지정 가능(padding-top, padding-right, padding-bottom, padding-left)
  • 크기, 백분율, auto 사용 가능
    • ex) padding : 5px 10px 15px 20px

margin 속성

  • 현재 요소 주변의 여백
  • 마진을 이용하면 요소와 요소 간의 간격 조절 가능
  • 각 방향별로 지정 가능(margin-top, margin-right, margin-bottom, margin-left)
  • 크기 : 너비나 높이 값을 px이나 cm같은 단위와 함께 수치로 지정
  • 백분율 : 박스 모델을 포함하고 있는 부모 요소를 기준으로 너비나 높이 값을 %로 지정>
  • auto : display 속성에서 지정한 값에 맞게 적절한 값을 자동으로 지정.
    • ex) margin : 10px
    ex) margin : 10px 20px
    ex) margin : 10px 20px 30px
    ex) margin : 10px 20px 30px 40px
