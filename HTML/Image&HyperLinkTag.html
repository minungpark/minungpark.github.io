앵커 : 한 페이지 안에서 점프하기.

<a>태그 : 하이퍼 링크, href 속성을 이용하여 링크 주소 입력
target속성 : 링크한 내용이 표시될 위치 지정(_blank : 새 창 or 새 탭)
target속성 : 링크한 내용이 표시될 위치 지정(_self : 기본값, 링크가 있는 화면에서 열림)
target속성 : 링크한 내용이 표시될 위치 지정(_parent : 프레임을 사용했을 때 링크 내용을 부모 프레임에 표시)
target속성 : 링크한 내용이 표시될 위치 지정(_top : 프레임을 사용했을 때 프레임을 벗어나 링크 내용을 전체 화면에 표시)
download속성 : 링크한 내용을 다운로드
rel속성 : 현재 문서와 링크한 문서의 관계를 알려줌
hreflang속성 : 링크한 문서의 언어 지정
type속성 : 링크한 문서의 파일 유형을 알려줌

앵커 사용법

임의의 태그에 id 속성을 추가한 다음,
a 태그의 href의 속성에 이동하고 싶은 태그에 적용된 id 속성을 적용해주면,
해당 링크를 누를시 적용된 id에 해당하는 태그가 있는 곳으로 점프한다.
ex)<h2 id="content2">
<a href="#content2">
